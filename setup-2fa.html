How to Scan

In Microsoft Authenticator:

  • Tap the + button
  • Select "Other account"
  • Point your camera at the QR code
Code Not Working?

Make sure your phone's time is set to automatic. To fix this:

  • iPhone: Settings → General → Date & Time → Set Automatically
  • Android: Settings → System → Date & Time → Use network time
