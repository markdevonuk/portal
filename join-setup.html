Setup Applicants Collection

FMS Prehospital Portal

Quick Setup

This utility will initialize the applicants collection in your Firebase database, creating the necessary structure for handling membership applications.

Using your existing Firebase project: fmsportal-dbfb3

Status: Ready to initialize

Collection Structure

The applicants collection will use the following schema:

{
  "firstName": "String",
  "surname": "String",
  "dob": "Date",
  "email": "String",
  "mobile": "String",
  "address": "String",
  "qualification": "String",
  "giftAid": "Boolean",
  "applicationDate": "Timestamp",
  "status": "String", // 'pending', 'approved', 'rejected'
  "paymentStatus": "String", // 'unpaid', 'paid'
  "paymentDate": "Timestamp" // Optional
}

Setup Complete!

The applicants collection has been successfully initialized.
